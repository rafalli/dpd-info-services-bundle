# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-16
### Added
- Initial stable release.
- Integration with DPD InfoServices API (SOAP) for `getEventsForCustomerV4` and `getEventsForWaybillV1`.
- `WaybillDeliveryTracker` for easy domain-level delivery status checks.
- Full Symfony 6.4 / 7.x / 8.x and PHP 8.1 - 8.4 support.
